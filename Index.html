The Psychometric Framework

This experience is built on an Evidence-Centered Design (ECD) model. Every task is engineered to elicit measurable evidence about the player. We measure four core domains: Natural Talents, Work-Life Personality, Core Motivators, and Communication Style. The goal is to create a dynamic, evidence-backed profile that evolves with gameplay, providing a nuanced portrait of the player's potential.

Visualizing the Connections

The power of this system is its interconnectedness. A single action can inform multiple aspects of the player's profile. Hover over a game task on the outer ring to see how it links to the various constructs we measure across different domains.

Publishing Your Application

This single-page application is self-contained. Here are two easy methods to deploy it to a live URL and connect it to a custom domain.

Method 1: GitHub Pages

  1. Save this file as `index.html`.
  2. Create a new public repository on GitHub.
  3. Upload the `index.html` file to the repository.
  4. In the repository's settings, go to the "Pages" section.
  5. Select the `main` branch as the source and click "Save".
  6. Your site will be live at `your-username.github.io/repository-name/`.
  7. To use a custom domain, add your domain name here and configure the CNAME/A records with your domain registrar.

Method 2: Netlify Drop

  1. Save this file as `index.html`.
  2. Create a free account at `netlify.com`.
  3. Go to the "Sites" section of your dashboard.
  4. Simply drag and drop the `index.html` file into the browser window.
  5. Netlify will instantly deploy your site to a random URL.
  6. You can change the site name or add a custom domain in the "Domain settings" for your new site.
